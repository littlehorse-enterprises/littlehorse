# Littlehorse 1.1.2

The LittleHorse v1.1.2 release is a focused patch that strengthens server stability and security. It eliminates a resource leak by properly closing iterators on the read-only store and corrects access-control validation so permissions are enforced accurately for specific resources.

### Server
- Fixed a resource leak by ensuring iterators on the read-only store are properly closed, improving long-running server stability and reducing memory pressure. #2364
- Corrected validation of permitted ACL actions for specific resources, ensuring access-control policies are enforced accurately and securely. #2367
- Prevent orphaned ThreadRuns when exceeding max ThreadRun limit #2360
