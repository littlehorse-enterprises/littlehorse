# Littlehorse 1.1.1

LittleHorse v1.1.1 includes targeted stability improvements to the server. This release addresses a race condition that could corrupt metadata internal cache and fixes incorrect task attempt tracking during timeout scheduling.

### Server
- Fixed a race condition that could occur when concurrently updating metadata objects. This affected only the in-memory cache, not the persisted data. (#2291)
- Fixed missing attempt numbers when scheduling task timeouts, ensuring timeouts are correctly attributed to the right task attempt. (#2340)
